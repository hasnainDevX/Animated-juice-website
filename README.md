# 🧃 Animated Juice Website

A playful, animated landing page for a juice/drinks fictional brand — built with **React**, **Tailwind CSS**, and **GSAP** for smooth, engaging motion design.

🔗 **Live Site:** [animated-juice-website.vercel.app](https://animated-juice-website.vercel.app)

---

## 🎯 What This Project Demonstrates

- Building animated, motion-heavy UIs with **GSAP** inside a React app
- Component-based architecture with React
- Responsive, utility-first styling with Tailwind CSS

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Framework | React |
| Styling | Tailwind CSS |
| Animation | GSAP |
| Deployment | Vercel |
